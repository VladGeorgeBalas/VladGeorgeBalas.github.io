# Homepage

## Scurta descriere!
Aceasta pagina este un fel de Hub pentru toate proiectele si materialele mele, de pe contul de git de facultate.
Pagini de interes sunt cele dedicate materiilor de facultate cat si cele dedicate proiectelor idependente.

## Motivatie?
Nici eu nu mai imi gasesc paginile de github printre ce am pe cont, paginile de acest fel sunt bune ca mod de cautare printre ele. Trebuie
vazuta aceasta pagina ca un fel de meta-index, care pe langa liste clare cu pagini de github are si un readme superior celui de pe pagina
care explica mai in detaliu ce am vrut sa fac, deoarece tind sa uit.

## Ce nu e aceasta pagina
Nu este un blog sau un site personal, si nici un fel de CV interactiv, deci pot lipsi detalii complete despre ce si cum se intampla prin 
paginile prezentate. Pagina asta e aproape imposibil de gasit neintentionat, deci presupun ca daca citesti poti sa ma si contactezi cu
intrebari.

**Distratie placuta in a gasi ce cauti!!**
